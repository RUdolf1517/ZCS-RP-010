Формат Excel файла:

Файл должен содержать следующие столбцы (в первой строке - заголовки):

  1. ФИО - Полное имя ученика (обязательно)
  2. Класс - Класс ученика, например "7А" (обязательно)
  3. Кл. руководитель - ФИО классного руководителя (обязательно)
  4. Достижения - Список достижений через точку с запятой (опционально)

Пример данных:

ФИО Класс Кл. руководитель Достижения
Иванов Иван Иванович 7А Петрова Анна Сергеевна Олимпиада по математике; Конкурс чтецов
Петрова Мария Александровна 8Б Сидоров Михаил Петрович
Важно:
  • Первая строка файла должна содержать заголовки столбцов
  • Обязательные поля не должны быть пустыми
  • Ученики с одинаковым ФИО и классом будут пропущены
  • Достижения будут автоматически преобразованы в стандартный формат
